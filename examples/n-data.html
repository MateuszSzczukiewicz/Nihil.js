Nihil.js n-data Examples

These examples demonstrate the n-data directive, which initializes a component's data scope. Check the console for logs.

Component with Inline Object Data

This div has an n-data attribute with an inline JavaScript object.

n-data="{ message: 'Hello from Nihil.js!', count: 10 }"

Component with Empty Object Data

This div has an n-data attribute with an empty JavaScript object.

n-data="{}"

Component with Data from a Function Call

This div's n-data attribute calls a JavaScript function (defined in shared-scripts.js).

n-data="getComponentData()"

Component with n-data attribute but no value

This should trigger a warning in the console.

n-data

Component with n-data attribute and empty string value

This should also trigger a warning in the console.

n-data=""

Component with Syntax Error in n-data

This should trigger an error in the console during evaluation.

n-data="{ message: 'Unterminated string }"
